\f0\fs24 \cf0 ## Purpose of our program:\
\
The purpose of our program to take in a root folder to be traversed and a number of mapper processes and to return a count of all alphabetical characters that appear in text files within the folders. The number of mapper processes determine how many processes will be spawned to complete the task of the function. \
\

## How to compile the program\
\
To compile the program simply type \'91make\'92 into the command line and run it, this will produce an executable, \'91mapreduce\'92. To run the program enter into the command line: \'91./mapreduce folderName #mappers\'92 where folderName is the name of the root folder to be traversed and #mappers is the number of mapper processes you would like to execute the program with. \

## What exactly our program does\
\
Our program takes a root folder and traverses through the directories within it, accessing text files. The program will then read the text files and store and return a count of all the alphabetical letters within the text files in the all directories within the root folder. This is done by process communication through pipes.\
